# Export Guide

Learn how to export your demand letters to Word and PDF formats.

## Overview

Export options:
- **Word (.docx)** - For further editing in Microsoft Word
- **PDF** - For final delivery and archival

Both formats preserve:
- Text content and formatting
- Basic styling (bold, italic, headings)
- Page layout
- Firm branding (if configured)

## Exporting to Word

### Quick Export

1. Open your finalized demand letter
2. Click "Export" button in toolbar
3. Select "Export to Word"
4. File downloads automatically
5. Open in Microsoft Word or compatible editor

### Word Export Features

**What's included:**
- Full letter content
- Basic text formatting
- Headings and paragraphs
- Lists (bullet and numbered)
- Page breaks
- Firm letterhead (if configured)

**Editable in Word:**
- All text
- Formatting
- Layout
- Headers and footers

**Use Word export when:**
- You need to make manual edits
- Adding graphics or images
- Customizing formatting
- Sharing for attorney review before sending

### Word Formatting

**Preserved:**
- Bold, italic, underline
- Font sizes (headings vs body)
- Paragraphs and spacing
- Lists and indentation

**Not preserved:**
- Some advanced colors
- Custom fonts (uses system defaults)
- Complex layouts

**Recommended workflow:**
1. Export to Word
2. Open in Word
3. Apply your firm's standard formatting
4. Add any custom elements
5. Save or convert to PDF for delivery

## Exporting to PDF

### Quick Export

1. Open your finalized demand letter
2. Click "Export" button in toolbar
3. Select "Export to PDF"
4. File downloads automatically
5. View in any PDF reader

### PDF Export Features

**What's included:**
- Locked, final-format letter
- All formatting preserved
- Professional appearance
- Ready for delivery
- Firm letterhead (if configured)

**Cannot be edited:**
- PDF is read-only
- Protects letter integrity
- Professional final format

**Use PDF export when:**
- Ready to deliver to opposing counsel
- Archiving final version
- Sharing for review (view-only)
- Email or postal delivery

### PDF Settings

**Default settings:**
- Letter size (8.5" x 11")
- Portrait orientation
- 1-inch margins
- Standard fonts

**Advanced options (click "Export Settings"):**
- Page size (Letter, Legal, A4)
- Orientation (Portrait, Landscape)
- Margins (narrow, normal, wide)
- Include page numbers
- Include headers/footers
- Watermark (Draft, Confidential)

## Batch Export

Export multiple letters at once:

1. Go to "Demand Letters" list
2. Select multiple letters (checkboxes)
3. Click "Export Selected"
4. Choose format (Word or PDF)
5. Choose individual files or single ZIP
6. Downloads complete

**Use batch export for:**
- End-of-month archiving
- Preparing multiple letters for review
- Creating backup copies

## File Naming

**Default naming convention:**
```
[Case Name] - Demand Letter - [Date].docx
```

Example: `Smith v Jones - Demand Letter - 2025-11-14.docx`

**Customize file names:**
1. Click "Export Settings"
2. Edit "File Name Template"
3. Use variables:
   - `{case_name}`
   - `{date}`
   - `{plaintiff}`
   - `{defendant}`
4. Example: `{plaintiff}_v_{defendant}_Demand_{date}`

## Export Quality

### Best Practices

**Before exporting:**
- [ ] Final review complete
- [ ] All edits finalized
- [ ] No placeholder text
- [ ] Numbers and dates verified
- [ ] Spelling checked
- [ ] Formatting consistent

**For professional results:**
- Use templates with proper formatting
- Keep formatting simple
- Avoid unusual fonts
- Test export before sending
- Review exported file

### Troubleshooting Export Issues

**Export fails:**
- Check internet connection
- Try different format
- Refresh page and retry
- Check browser allows downloads
- Contact support if persistent

**Formatting looks wrong:**
- Use simpler formatting in editor
- Avoid complex nested elements
- Check template formatting
- Try exporting to Word first, then PDF

**File won't open:**
- Verify you have appropriate software (Word, PDF reader)
- Check file downloaded completely
- Try different PDF/Word viewer
- Re-export if corrupted

**Letterhead not appearing:**
- Verify firm branding configured
- Check administrator settings
- Try different export format
- Clear cache and retry

## Delivery Methods

### Email Delivery

**Attach exported file:**
1. Export to PDF (recommended) or Word
2. Compose email to opposing counsel
3. Attach exported file
4. Include cover message
5. Send

**Best practices:**
- Use PDF for final demands
- Include deadline in email body
- CC your client if appropriate
- Keep copy of sent email
- Follow up if no response

### Print and Mail

**For postal delivery:**
1. Export to PDF
2. Print on firm letterhead
3. Review printed copy
4. Sign manually
5. Mail via certified mail (recommended)

**Printing tips:**
- Print test page first
- Use high-quality paper
- Ensure letterhead aligns correctly
- Sign in blue ink for distinction

### Document Management Systems

**Save to DMS:**
1. Export to appropriate format
2. Upload to your DMS
3. Tag with case number
4. Link to case file
5. Set appropriate permissions

## Export History

Track all exports:

1. Open demand letter
2. Click "Export History"
3. View list showing:
   - Export date and time
   - Format (Word, PDF)
   - Who exported
   - File size

**Use export history to:**
- Verify when letter was delivered
- Track different versions
- Audit trail for case file
- Confirm delivery dates

## Security

**Exported files are:**
- Clean (no hidden metadata beyond standard document properties)
- Virus-scanned
- Encrypted during download
- Stored temporarily (deleted after download)

**Recommendations:**
- Use secure email for sending
- Encrypt sensitive PDFs (use PDF software)
- Delete local copies after case closes
- Store in secure document management system

## Frequently Asked Questions

**Can I export before finalizing?**
Yes. Export drafts for review anytime.

**Which format should I use?**
PDF for final delivery, Word if further editing needed.

**Can I edit PDF after export?**
Not directly. Export to Word, edit, then convert to PDF.

**Are exports tracked?**
Yes. All exports logged in export history.

**Can I export to other formats?**
Currently only Word and PDF. More formats coming.

**Is there a file size limit?**
No practical limit. Typical letters are 50-200 KB.

**Can I customize the PDF formatting?**
Yes, through Export Settings and firm branding configuration.

**Do exports cost AI tokens?**
No. Exports don't use AI and have no additional cost.
