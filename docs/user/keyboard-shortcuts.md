## Tips for Using Shortcuts

### Learning Shortcuts

1. **Start with most-used actions**: Save, Generate, Export
2. **Learn gradually**: Add 2-3 new shortcuts per week
3. **Use tooltip hints**: Hover over buttons to see shortcuts
4. **Practice regularly**: Muscle memory develops over time

### Discovering Shortcuts

- Hover over buttons and menu items
- Press `?` to open help panel with shortcut reference
- Check this documentation for complete list
- Look for keyboard hints in menus

## Platform Differences

### Windows/Linux vs. Mac

Most shortcuts follow this pattern:
- `Ctrl` on Windows/Linux = `Cmd` on Mac
- `Alt` on Windows/Linux = `Opt` on Mac
- `Windows` key on Windows/Linux = `Cmd` on Mac

### Special Keys

- `Ctrl` = Control key
- `Cmd` = Command key (⌘) on Mac
- `Alt` = Alt key
- `Opt` = Option key (⌥) on Mac
- `Shift` = Shift key
- `Enter` = Enter/Return key
- `Esc` = Escape key
- `Delete` = Delete/Backspace key

## Troubleshooting Shortcuts

### Shortcut not working

**Possible causes:**
- Browser overrides the shortcut
- Focus not on correct element
- Operating system uses the shortcut
- Conflicting browser extension

**Solutions:**
1. Click in the appropriate area first
2. Check browser shortcuts don't conflict
3. Disable conflicting extensions
4. Try in different browser
5. Use mouse/button alternative

### Finding the right shortcut

Can't remember a shortcut?

1. Hover over the button (tooltip shows shortcut)
2. Press `?` to open help panel
3. Check this reference page
4. Use search (`Ctrl+K` / `Cmd+K`)
